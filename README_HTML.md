# ENSGSI12IS HTML Dashboard

## Overview
A comprehensive HTML dashboard showcasing the complete ENSGSI12IS project with all outputs, visualizations, and GitHub integration.

## Features
- **Modern UI**: Quantum-themed design matching ZI-US style
- **Interactive Navigation**: Tab-based navigation between sections
- **Live Graphs**: Direct display of all generated PNG graphs
- **File Download**: Direct links to all output files
- **Collaborator Showcase**: Team members with GitHub links
- **GitHub Integration**: Branch links and repository stats
- **Mobile Responsive**: Works on all device sizes

## Sections Included
1. **Project Overview** - Complete project description and key metrics
2. **Phase Outputs** - All generated files from 4 phases with download links
3. **Visualizations** - PNG graphs displayed directly in browser
4. **Collaborators** - Team members with GitHub profiles
5. **GitHub Repository** - Branch links and repository information
6. **Documentation** - MS Word integration guide and verification commands

## How to Use
1. Open `index.html` in any modern web browser
2. Use navigation tabs to switch between sections
3. Click on graphs to view full size (if available locally)
4. Use download links to get output files
5. Click GitHub badges to visit repository
